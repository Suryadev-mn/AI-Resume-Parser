# AI Resume Parser

An AI-powered Resume Parser built using Python, Streamlit, and NLP.

## Features
- PDF Resume Upload
- Name Extraction
- Email Extraction
- Phone Extraction
- Skills Detection
- Education Detection
- Experience Detection
- Resume Scoring
- Job Description Matching
- CSV Report Download

## Tech Stack
- Python
- Streamlit
- SpaCy
- PyPDF2
- Pandas

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
